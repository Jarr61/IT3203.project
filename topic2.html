Grid Basics

Grid is another layout system. It creates a grid-based layout system, also with rows and columns. This is used so developers can more easily create complex web layouts.

When should I use Grid?

  • Complex layouts with many sections
  • Photo or card galleries
  • Anything that needs rows or columns, eg. numbers or random lists.
